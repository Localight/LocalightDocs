Ngrok Setup Guide

Setting up the magical ngrok


Steps:

- Download ngrok: https://ngrok.com/
- Sign up for ngrok and get your auth token.
- Create a configuration file called ngrok.yml

Put the following code in that file, replacing every ~~INSTRUCTION~~ with your own information. (do not leave these in the file)

authtoken: ~~Your authtoken~~
http_proxy: "http://127.0.0.1:8888" ~~Include this line if you want to use charles~~
tunnels:
~~Tunnel name, such as juliantunnel~~:
proto: http
addr: ~~Port of your grunt server~~

Make sure that the file tabbed in correctly from copy/pasta!
I explain this file at the end of this document in more detail. For now, lets start the tunnel!

Open your terminal, and enter the following command from the directory your ngrok is in:

"./ngrok start ~~tunnel name~~ --config /~~path to your config.yml~~"

You will now be able to access your localhost from tunnelname.ngrok.io (no port number). It will automatically be tunneled to your localhost, at the port you specified.

Config.yml in greater detail:

authtoken: Specifies your authtoken with ngrok
http_proxy: Directs ngrok through charles, if you so wish. Otherwise, ngrok will tunnel past charles.
tunnels: You can define multiple tunnels here if you want.
tunnel name (also your url):
protocol: Tells ngrok to use http, instead of other types of web requests. Ngrok does it all!
addr: Specifies the internal port of the development server you want ngrok to route to.
